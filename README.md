## âœ¨ Key Features

This Landing Page is crafted with attention to detail and modern web aesthetics:

### ğŸ¨ Visuals & Animations
- **Dynamic Background Orbs:** The background gradient colors (`orb1`, `orb2`) change dynamically based on the active slide, creating a glowing ambient effect.
- **Glassmorphism UI:** Modern glass-effect cards overlaid on product images for a premium look.
- **Custom Keyframe Animations:** 
  - `fadeInUp` for text elements.
  - `zoomIn` for the main slider entrance.
  - `spin` animation for decorative vector rings.
- **Spinning Star Element:** A subtle rotating star SVG adding life to the typography.

### ğŸ› ï¸ Functional Components
- **Auto-Play Slider:** A robust image slider that cycles automatically every 5 seconds.
- **Interactive Controls:** Manual Next/Previous navigation buttons with hover effects.
- **Dark/Light Mode Support:** Native Tailwind support for dark mode (`dark:bg-[#0f0f0f]`), making the floral colors pop in low light.
- **Responsive Layout:** Optimized for Mobile, Tablet, and Desktop viewing experiences.

## ğŸ’» Getting Started

Follow the instructions below to run this project locally on your machine.

### Prerequisites
Make sure you have **Node.js** installed.

### Installation Steps

**1. Clone the repository**
```bash
git clone https://github.com/NahidAhmed-12/flower-shop.git


**2. Navigate to project directory**
```bash
cd flower-shop
```

**3. Install Dependencies**
```bash
npm install
# or if you use yarn
yarn install
```

**4. Start the Development Server**
```bash
npm run dev
# or
yarn dev
```

The application will open automatically at `http://localhost:5173` (or your default Vite port).
